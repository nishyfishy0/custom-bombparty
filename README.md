# Custom BombParty Clone
A text-baseed version of the web game BombParty (which can be played on html://jklm.fun) made entirely in Python, with added features.


## Gamemodes
### Singleplayer
Time attack; Answer as many valid words as possible without losing all your lives and within your allotted time!


### Multiplayer
**Hot Potato**
Pass the bomb around to each player! Try not to let the bomb explode on you by answering as fast as possible!

**Time Attack**
Everyone has their own timer, so try not to run out of time! (One Life)

**Co-op**
Sharing is caring! Everyone shares the timer *and* lives! Work together to get as many words as possible before running out of lives!


## How To Play
1. Clone this repository or download the script.
2. Ensure you have your dictionary folders structured as:
   `custom bombparty/dict.txt`
   `custom bombparty/morphemes.txt`
3. Run the script via your terminal.
